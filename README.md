# Lil' Blog
---

This repo is meant to document not only the versions of the blog I'll create for
fun, but the whole experience. I'll talk more about it on the blog itself!

## Plans
> List of things that I want to do at some point on the website

- [ ] Post system.
    - A simple system for posts, where there will be a file (eg. `.txt`) that
    represents a post and the page loads it. Features: Title, content, image(s)
    rendering, hyperlinks and file attaching in general. And comments for the
    posts
- [ ] Chat room.
    - A simple chat system(maybe third party, still thinking about it) where
    people will have their own profile on the site, with picture and name
    displayed on the side of their messages.
- [ ] Affiliate links.
    - My plan consist mostly on me creating a secure web, of some sort, with
    this project I hope to inspire the people around me to make their own too,
    so we can create a community.
